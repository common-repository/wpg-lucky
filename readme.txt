Random jump to any one wpg.im bbs members's blog.

== Description ==

Random jump to any one wpg.im bbs members's blog, If you want jump to your blog, you must join wpg.im bbs and submit your blog here: http://wpg.im/lucky/IFeelLucky

随机跳转到任何一个wpg.im论坛成员的博客, 如果你想跳转到您的博客, 你必须加入wpg.im论坛并通过 http://wpg.im/lucky/IFeelLucky 这个链接提交您的博客.

== Installation ==

(1) unzip the wpg-lucky.zip file in /wp-content/plugins

(2) Active the plugin 

(3) Add the widget, setup the plugin under administation panel 'options/WPG-Lucky'.
